# 🏨 Hostel Management & Complaint System

A web-based **Hostel Management & Complaint System** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
The system helps students submit complaints and requests while hostel authorities manage them efficiently.

---

## 📌 Features

- Complaint submission and tracking  
- Leave request management   
- Emergency alert system  
- Hostel room management  
- Role-based access control

---

## 👥 User Roles

**Student**
- Submit complaints  
- Apply for leave  
- Register visitors  
- Request medical support  
- Track request status  

**Warden**
- Review complaints  
- Approve leave and visitor requests  
- Assign complaints to staff  

**Admin**
- Manage users  
- Manage hostel rooms  
- Maintain system data

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Himanshuagrawal2003/Minor-Project.git
```

Install dependencies

```bash
cd frontend
npm run dev
```

```bash
cd backend
npm install
```

Run the project

```bash
npm start
```
